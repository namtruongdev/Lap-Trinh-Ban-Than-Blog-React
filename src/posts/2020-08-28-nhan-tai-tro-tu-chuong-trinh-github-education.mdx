Github Education là một chương trình giáo dục được Github và các công ty công nghệ lớn trên thế giới liên kết lại để tài trợ cho toàn bộ mọi người đang là sinh viên và giảng viên tại các trường đại học trên thế giới.

## Mục đích của gói tài trợ

\- Giúp sinh viên có thể phát triển năng lực bản thân một cách tốt hơn, tự do sáng tạo, phá cách. Chúng ta chỉ cần học, việc còn lại cứ để Github lo. Lo như nào? Đó là họ cho chúng ta quyền truy cập miễn phí vào các công cụ và dịch vụ hàng đầu trong ngành như: Heroku, Amazon Web Service, Mongodb, Azure... Chúng ta còn được miễn phí đăng ký tên miền .com, .tech và nhiều thứ xịn xò khác nữa, nhiều quá mình không kể hết được. Các bạn có thể xem chi tiết các công cụ và dịch vụ được Github tài trợ theo [link này](https://education.github.com/pack) nhé.

\- Cũng giống như sinh viên, giảng viên có thể truy cập vào hàng trăm công cụ và dịch vụ đầu ngành để giảng dạy.

## Giới thiệu về Git và Github

Github với những người yêu lập trình chắc không còn xa lạ gì nhưng mình vẫn xin nhắc lại cho một số bạn không biết nó là gì?

<Quote>
  GitHub là một dịch vụ cung cấp kho lưu trữ mã nguồn Git dựa trên nền web cho
  các dự án phát triển phần mềm.
</Quote>

Vậy Git là gì?

<Quote>
  Git là phần mềm quản lý mã nguồn phân tán được phát triển bởi Linus Torvalds
  vào năm 2005, ban đầu dành cho việc phát triển nhân Linux
</Quote>

Nói một cách dễ hiểu là chúng ta dùng Git để quản lý mã nguồn (source code), sau khi code ra sản phẩm thì chúng ta cần có một nơi để lưu trữ nó, lúc đó ta cần đến Github.

Cách sử dụng Git và Github như nào mình sẽ không đề cập trong bài này vì mục đích của bài này không phải giúp các bạn học cách sử dụng chúng mà là giúp các bạn nhận được gói tài trợ từ chương trình Github Education.

## Nhận tài trợ từ gói Github Student Developer Pack

\- Chuẩn bị:

\+ 1 tài khoản Github (đăng ký bằng Email edu của trường đại học cấp hoặc nếu có tài khoản trước rồi thì các bạn chỉ cần vào cài đặt add thêm cái Email edu vào là được). Nếu đăng ký tài khoản Github mới, tạo xong bạn nhớ đăng nhập vào Email edu để xác minh tài khoản nha (Lúc đăng ký xong, Github sẽ gửi 1 tin nhắn xác minh về Email đăng ký).

\+ 1 ảnh chụp cận cảnh, rõ nét mặt trước của thẻ sinh viên.

### Bước 1

\- Truy cập vào [Github Education](https://education.github.com/).

### Bước 2

\- Chọn **Github Student Developer Pack** (Nếu là giảng viên thì chọn **Github Teacher Toolbox**).

```textile

help me better in developing open source software in university.

I love open source and I love sharing my knowledge with everyone around me, I will use
some sponsored services like Heroku, AWS ... to host my projects.

Thanks Github team.
```
