Chào các bạn, hôm nay mình sẽ tổng hợp các Package cho Sublime Text 3 giúp bạn đẩy nhanh quá trình phát triển web của bạn phần nào đó.

So với các code editor khác thì Sublime Text là một code đáng dùng.

Nó nổi trội về mặt nhẹ, đơn giản, dễ sử dụng rất thích hợp cho bạn nào mới làm quen đến lập trình hoặc thích sự đơn giản.

## Chủ đề

Chủ đề mình đang dùng là <a title="Seti_UI" target="_blank" rel="noopener" href="https://packagecontrol.io/packages/Seti_UI">Seti_UI</a>. Các bạn có thể tham khảo chủ đề này.

## Package

### <a rel="noopener" title="AdvancedNewFile" target="_blank" href="https://packagecontrol.io/packages/AdvancedNewFile">AdvancedNewFile</a>

Plugin này cho phép bạn tạo tập tin nhanh hơn trong một dự án.

### <a rel="noopener" title="Alignment" target="_blank" href="https://packagecontrol.io/packages/Alignment">Alignment</a>

Plugin cho phép bạn dễ dàng căn chỉnh css thẳng hàng theo dấu hai chấm.

### <a rel="noopener" title="AutoFileName" target="_blank" href="https://packagecontrol.io/packages/AutoFileName">AutoFileName</a>

Plugin tự động điền tên tệp khi bạn thêm một link nào đó từ trong thư mục của bạn mà không nhớ tên rõ tên file.

### <a rel="noopener" title="BracketHighlighter" target="_blank" href="https://packagecontrol.io/packages/BracketHighlighter">BracketHighlighter</a>

Plugin đánh dấu và làm nổi bật các dấu ngoặc, giúp bạn dễ soi code hơn.

### <a rel="noopener" title="Emmet" target="_blank" href="https://packagecontrol.io/packages/Emmet">Emmet</a>

Viết HTML siêu nhanh chỉ bằng nút tab hoặc các cú pháp viết tắt.

Cải tiến cho thanh sidebar của Sublime Text 3 với nhiều lựa chọn và chức năng hơn.
